About InfoGuard

Version

1.0.0

What is InfoGuard?

InfoGuard is a browser extension that uses Google's Gemini 2.0 Flash AI to detect deepfakes, AI-generated content, and media manipulation in real-time while you browse.

How it Works

  1. Detects images and videos on web pages and social media
  2. Uses Gemini AI visual reasoning to identify artifacts and inconsistencies
  3. Cross-references claims with trusted fact-checking databases
  4. Provides a credibility score from 0-100%
  5. Displays visual badges on media items for quick assessment

Privacy & Security

  • All authentication uses OAuth 2.0 - your password is never stored
  • Media is sent to Google's secure API endpoints for analysis
  • Analysis results are not stored on our servers
  • You control what gets analyzed via the extension settings
  • Open source - audit our code on GitHub
